# 📡 STOCKSONAR 설치 가이드 (비개발자용)

서버 없이 **전부 무료**로 운영하는 구조입니다. 코딩 없이 클릭만으로 설치할 수 있습니다.

```
GitHub Actions (무료 로봇)          GitHub Pages (무료 호스팅)
20분마다 자동 실행                   https://내아이디.github.io/stocksonar/
 ├ GlobeNewswire 뉴스 수집
 ├ 소송·재탕·비상장 필터링
 ├ yfinance 시세·시총·유통주식
 ├ Claude AI 한국어 분석             텔레그램 (선택)
 └ data.json 갱신 ──────────────→   고득점 신호 알림
```

유일한 비용은 Claude API뿐입니다 (하이쿠 모델 기준 뉴스 1건당 약 3~7원, 하루 100~200건 분석 시 **월 1~3만원** 수준).

---

## 1단계. Claude API 키 발급 (10분)

1. https://console.anthropic.com 접속 → 가입
2. 좌측 **Billing** → 결제 카드 등록 → $10 정도 크레딧 충전 (한 달 이상 사용량)
3. 좌측 **API Keys** → **Create Key** → 이름 아무거나 → 생성된 `sk-ant-...` 키를 복사해 메모장에 보관

> ⚠️ 이 키는 비밀번호와 같습니다. 절대 공개 장소에 붙여넣지 마세요. (아래 5단계의 Secrets에만 입력)

## 2단계. GitHub 가입 + 저장소 만들기 (5분)

1. https://github.com 가입
2. 우측 상단 **+** → **New repository**
3. Repository name: `stocksonar` / **Public** 선택 (Public이어야 Actions·Pages가 무제한 무료)
4. **Create repository** 클릭

## 3단계. 파일 업로드 (5분)

1. 받으신 `stocksonar-pipeline.zip` 압축을 풉니다. 내부 구조:
   ```
   pipeline.py              ← 수집·분석 로봇
   requirements.txt         ← 필요한 프로그램 목록
   .github/workflows/scan.yml  ← 20분마다 실행 스케줄
   docs/index.html          ← 사이트 화면
   docs/data.json           ← 분석 데이터 (시드 포함)
   설치가이드.md
   ```
2. GitHub 저장소 화면에서 **Add file → Upload files**
3. 압축 푼 폴더 안의 **모든 파일과 폴더를 통째로 드래그** (`.github` 폴더 포함 — 폴더째 끌어다 놓으면 구조가 유지됩니다)
4. 아래 **Commit changes** 클릭

> 💡 `.github` 폴더가 안 보이면: Windows 탐색기 → 보기 → "숨긴 항목" 체크

## 4단계. API 키 등록 (3분)

1. 저장소 → **Settings** → 좌측 **Secrets and variables → Actions**
2. **New repository secret** 클릭
3. Name: `ANTHROPIC_API_KEY` / Secret: 1단계에서 복사한 `sk-ant-...` 키 → **Add secret**

## 5단계. 사이트 켜기 — GitHub Pages (2분)

1. 저장소 → **Settings** → 좌측 **Pages**
2. Source: **Deploy from a branch**
3. Branch: `main` / 폴더: `/docs` → **Save**
4. 1~2분 후 `https://내아이디.github.io/stocksonar/` 에서 사이트가 열립니다 (시드 데이터 16건이 먼저 보입니다)

## 6단계. 로봇 첫 실행 (2분)

1. 저장소 → **Actions** 탭 → 워크플로 사용 동의 버튼이 보이면 클릭
2. 좌측 **STOCKSONAR scan** → 우측 **Run workflow** → **Run workflow**
3. 1~3분 후 초록색 체크 ✅ 가 뜨면 성공 — 이후엔 미국 장시간 동안 20분마다 자동 실행됩니다
4. 사이트를 새로고침하면 실제 뉴스 분석이 쌓이기 시작합니다

---

## (선택) 텔레그램 알림 연결 — 10분

SONAR 70점 이상 호재가 잡히면 텔레그램으로 바로 받아볼 수 있습니다.

1. 텔레그램에서 `@BotFather` 검색 → `/newbot` → 봇 이름·아이디 입력 → 발급되는 **토큰** 복사
2. 만든 봇을 검색해 **대화 시작(Start)** 버튼 클릭, 아무 메시지 1개 전송
3. 브라우저 주소창에 `https://api.telegram.org/bot토큰여기에/getUpdates` 입력 → 화면에서 `"chat":{"id":123456789` 숫자를 찾음 (이게 chat_id)
4. 4단계와 같은 방법으로 Secrets 2개 추가:
   - `TELEGRAM_BOT_TOKEN` = 봇 토큰
   - `TELEGRAM_CHAT_ID` = chat id 숫자

## 운영 팁

| 하고 싶은 것 | 방법 |
|---|---|
| 알림 기준 점수 변경 | `scan.yml`의 env에 `ALERT_MIN_SCORE: "80"` 추가 (GitHub에서 연필 아이콘으로 직접 수정 가능) |
| 실행 주기 변경 | `scan.yml`의 `*/20` 을 `*/30`(30분) 등으로 수정 |
| 더 똑똑한 분석 | Secrets 대신 `scan.yml` env에 `MODEL: "claude-sonnet-4-6"` 추가 (비용 약 4배) |
| 잠시 중지 | Actions 탭 → STOCKSONAR scan → 우측 ⋯ → **Disable workflow** |
| 비용 확인 | console.anthropic.com → Usage |

## 알아두실 점

- **지연**: GitHub Actions 무료 스케줄은 몇 분씩 밀릴 수 있습니다. "수초 내 초고속 알림"이 필요해지면 그때 유료 전환(RTPR 등)을 검토하면 됩니다.
- **시세 누락**: 무료 yfinance가 가끔 차단되면 해당 뉴스는 건너뜁니다 (다음 뉴스부터 정상).
- **공개 저장소**: 코드는 누구나 볼 수 있지만 API 키는 Secrets에 있어 안전합니다.
- **면책**: 사이트 하단 고지대로, AI 분석은 오류가 있을 수 있고 투자 자문이 아닙니다.

## 문제가 생기면

Actions 탭 → 실패한 실행(❌) 클릭 → 로그 화면을 캡처해서 Claude(저)에게 보여주세요. 바로 고쳐드립니다.
